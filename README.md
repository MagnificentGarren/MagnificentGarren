## 🚀 Highlighted Projects

### 🤖 [Multi-Agent Strategic Research Assistant](https://github.com/MagnificentGarren/multi_agent_researcher)
An enterprise-grade, local multi-agent research and compilation engine that automates technical document drafting through iterative critique gates.
* **Architecture:** Built with **LangGraph** to manage dynamic state routing across 4 distinct, cooperative agents (Researcher, Verifier, Summarizer, Critic).
* **Performance Optimization:** Powered by **Gemini 2.5 Flash** with custom token-saving context mitigation guardrails to prevent history bloat across long loop cycles.
* **Infrastructure:** Backed by a fault-tolerant **FastAPI** server using Server-Sent Events (SSE) to stream real-time agent node telemetry directly into a sleek **Next.js 14** and **Tailwind CSS** dashboard dashboard.

### 🌤️ [PyQt5 Async Weather Engine](https://github.com/MagnificentGarren/WeatherApp)
A responsive, desktop GUI utility tool providing real-time global weather streaming, built to explore asynchronous desktop request architectures.
* **Core Stack:** Built using **Python**, **PyQt5** (QtWidgets layout engineering), and the **OpenWeather API**.
* **Error Architecture:** Designed with strict decoupled exception handling and fallback mapping paths to handle `HTTP 4xx/5xx` server anomalies, network dropped pipes, and missing API configurations safely.
* **Clean UI Stylesheets:** Leveraged embedded styling syntax (`QSS`) to map custom layouts, typographical weights, and responsive dynamic display emoticons matching standard operating system font standards.
